These files were removed from the distributable Skill packages during the 2026-07-10
action-first rewrite. They remain available to maintainers and eval authors, but runtime
agents do not load them automatically because none owns an execution decision.

| File | Why archived |
| --- | --- |
| `worked-example-project-optimization.md` | Historical broad-audit example; superseded by the bounded P1 playbook. |
| `worked-examples.md` | Explanatory examples whose rules already live in specialist mechanics and P1-P6 gates. |
| `templates-review-and-governance.md` | Report/governance templates outside the runtime implementation loop. |

Do not copy an archived file back into a package unless a current playbook links it at a
specific step and an eval shows that loading it changes repository behavior rather than
only increasing answer length.
